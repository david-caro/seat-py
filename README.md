##Getting Started

**Dependencies**

* [PyYAML][3]

[3]: http://pyyaml.org/wiki/PyYAML

**Installation**
    
    $ git clone https://github.com/stackd/seat-py.git
    $ cd seat-py
    $ python setup.py install
